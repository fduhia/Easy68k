RichEditPlus
RichEditPlus adds Vertical Scroll events to RichEdit.


To install RichEditPlus in Borland Builder 6.0;

1.  Open dclRichEditPlus.bpk using the Borland C++ Builder IDE.

2.  Click the Install button in the Package - dclRichEditPLus.bpk window.

3.  Copy RichEditPlus.h to Borland\CBuilder6\Include folder.



After installation the RichEditPlus icon should be visible in the Win32 tab of the IDE.

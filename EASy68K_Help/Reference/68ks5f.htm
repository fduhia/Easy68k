CLR Instruction


This instruction (CLeaR) clears an operand specified with an effective address.

ADDRESS METHODS: Dn, (An), (An)+, -(An), x(An), x(An,xr.s), x.w, x.l

DATA LENGTH: Byte, word, longword
FLAGS: X - U
N - 0
Z - 1
C - 0
V - 0

SYNTAX: CLR <ea>

EXAMPLE CODE:   

   CLR     D0   clear the low word of D0 to zero
